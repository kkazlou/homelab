# Homelab

## Pre-requirements
Python 3.9 or newer installed

```bash
python3 --version
```

## Installing Ansible
```bash
python3 -m pip install --user ansible
```

## Upgrading Ansible
```bash
python3 -m pip install --upgrade --user ansible
```

## Confirming your installation
```bash
ansible --version
```
